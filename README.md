# Controla.AI

Monorepo com **frontend** (Vite + React + Tailwind) e **backend** (Fastify + Drizzle + PostgreSQL).

## Desenvolvimento

- **Backend:** `cd backend && npm install && npm run dev` (porta 3333). Configure `.env` a partir de `.env.example`.
- **Frontend:** `cd frontend && npm install && npm run dev` (porta 5173).
- **Banco:** `cd backend && npm run db:push && npm run db:seed`
- **Pacote extenso de dados (conta leonardosena1010@hotmail.com):** `cd backend && npm run db:seed:rich` (requer usuário já cadastrado)
